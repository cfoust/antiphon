# 網頁展示

[antiphon.dev/demo.html](https://antiphon.dev/demo.html) 是完整的引擎——同一顆 Rust DSP 核心編譯成 WebAssembly，渲染同一個房間，並透過瀏覽器的攝影機進行頭部追蹤（以 MediaPipe 取代原生追蹤器）。它不是影片，也不是近似品；引擎的原生與 wasm 建置，在 CI 等級的一致性測試中被驗證為逐位元組相同。

與應用程式不同之處：

- **代理是照劇本走的。** 四個角色會演完一段擬真的工作過程——工具滴答、低語、完成回報——而不是你真實的工作階段。
- **沒有回話（talk-back）。** 回覆流程需要在你的機器上執行常駐程式。
- **校正流程有所簡化**，但「保持不動」的引導步驟是一樣的。

在安裝任何東西之前，這是評估空間渲染最誠實的方式：如果戴上耳機後，聲音對你而言無法外部化，請在引導流程中調整貼合滑桿——若仍然不行，應用程式也不會有所不同。

請使用 Chrome 或 Safari，必須戴耳機，攝影機可有可無（你可以拒絕追蹤，改用滑鼠來轉動視角）。
