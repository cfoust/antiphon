# 開始使用

你已經[安裝好應用程式](./install.md)，也接上了一個代理。以下是 Antiphon 日常運作的樣子。

## 房間

主視窗是一張俯視雷達圖：你是正中央的瞳孔，代理則是環坐在你四周的圓點。拖曳代理即可移動它——聲音會跟著走，你也會從新位置聽到一聲輕柔的聲納脈衝。座位在重新啟動後仍會保留。

右側的清單列出每個工作階段：狀態、最後一則訊息，以及等待指示。將游標停在某一列上，就會在房間裡聚焦該代理。讓某個代理小睡（月亮圖示）可以把它移出這個世界——它仍會持續接收更新，喚醒後會從原處重新加入。

角落的眼睛按鈕是總開關。閉上的眼睛：釋放攝影機、輸出靜音、視窗轉暗。睜開的眼睛：持續觀看與聆聽。

## 你會聽到什麼

每個代理都有自己的音樂身分——一組以它的提示音為根音的和弦：

- **工具滴答。** 每次工具呼叫，都會撥響該代理下行和弦的下一個音。不用看，你也知道誰在忙。
- **工作嗡鳴。** 埋首於長任務的代理，會沉入音域底部一段低沉的三音機械嗡鳴。
- **等待綻放。** 完成工作、等著找你的代理，會在你的一側耳中緩慢醞釀出小七和弦的綻放——以分鐘計，而非毫秒。它從不強求；它只是累積。
- **語音。** 代理完成某件事時會親口說出來——一兩句話，用它自己的聲音，從它的座位傳來。

## 閉上眼睛

網路攝影機會追蹤你的頭部姿態與眼睛狀態（全部在裝置端完成——沒有任何資料離開你的電腦）。

睜眼時，語音會先被保留；你只會聽到環境音層。**閉上眼睛，房間便浮現**：一段短短的淡入之後，每個代理都恰好位在它的圓點所在之處。把頭轉向其中一個，停住一拍——一記輕柔的聲峰標記出鎖定，若它有消息，便會送上它的摘要。再轉向下一個聲音；如此往復。這是用耳朵完成的分流，只需幾秒鐘。

睜開眼睛，世界隨之退去。若有代理說了話，回覆框已在等候：輸入（或口述）後按下 Enter，你的話便會注入該代理的工作階段。

## 設定

齒輪按鈕會在視窗內開啟設定：

- **貼合（Fit）** — 一支頻率縮放滑桿，讓 HRTF 貼合你的耳朵；調整時會有一段循環播放的語音從正前方傳來。把它調到聲音聽起來最像*在你頭外*的位置，設定一次即可。
- **追蹤** — 攝影機的選擇與重新校正。
- **語音** — TTS 供應商、API 金鑰，以及個別語音的開關。
- **語言** — English、Русский、简体中文、繁體中文。
