# 隱私

Antiphon 是本機優先的應用程式。沒有帳號，沒有遙測，沒有分析統計——而所有可能敏感的部分，
從設計上就留在你的機器上。

## 攝影機

頭部追蹤完全在裝置上執行：原生應用程式把畫面交給 Apple 的 Vision 框架，網頁展示交給瀏覽器裡的
MediaPipe。影片從不錄製、從不儲存、從不傳輸——畫面在記憶體中分析後即被丟棄。當你闔上
Antiphon 的眼睛（選單列開關），擷取工作階段徹底停止，攝影機指示燈熄滅。

## 什麼會離開你的機器

只有一件事，而且只在你主動開啟時發生：**語音合成**。如果在「設定 → 聲音」裡加入了
ElevenLabs 或 OpenAI 的 API 金鑰，代理播報的*文字*會傳送給該供應商合成語音，適用其
隱私政策。合成的音訊快取在本機（`~/.antiphon/tts-cache`），重複的台詞不會再次傳送。
如果使用內建的 macOS 聲音，則沒有任何資料離開你的機器。

`antiphond` 常駐程式只監聽 `127.0.0.1`——從網路上無法存取。代理工作階段的播報從外掛到
常駐程式再到應用程式，全程走 localhost。

## 磁碟上有什麼

Antiphon 儲存的一切都在 `~/.antiphon`：

- `config.json` — 設定，包括 API 金鑰（檔案權限 `0600`）
- `agents.json` — 代理註冊表（工作階段 id、角色、座位）
- `tts-cache/` — 合成的音訊
- `antiphond.log` — 常駐程式的診斷紀錄，每次執行清空

刪掉這個目錄，Antiphon 就把你忘了——見[解除安裝](./install.md#解除安裝)。

## 這個網站

網站是靜態的（GitHub Pages）。下載按鈕會查詢 GitHub 的公開 API 來找到最新版本；網頁
展示完全在你的瀏覽器裡執行，不向任何地方傳送資料。
