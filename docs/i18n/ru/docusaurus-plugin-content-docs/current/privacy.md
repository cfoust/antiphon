# Конфиденциальность

Antiphon — локальное приложение. Никаких аккаунтов, телеметрии и аналитики, а всё
потенциально чувствительное спроектировано так, чтобы оставаться на вашей машине.

## Камера

Трекинг головы работает целиком на устройстве: нативное приложение передаёт кадры
фреймворку Apple Vision, веб-демо — MediaPipe в вашем браузере. Видео никогда не
записывается, не сохраняется и не передаётся — кадры анализируются и отбрасываются в
памяти. Когда вы закрываете глаз Antiphon (переключатель в строке меню), сессия захвата
останавливается полностью, и индикатор камеры гаснет.

## Что может покинуть вашу машину

Одна вещь, и только если вы сами это включили: **синтез речи**. Если добавить ключ
ElevenLabs или OpenAI в «Настройки → Голоса», *текст* реплик агентов отправляется этому
провайдеру для озвучивания — по его политике конфиденциальности. Синтезированный звук
кэшируется локально (`~/.antiphon/tts-cache`), поэтому повторяющиеся реплики не
отправляются заново. Если оставить встроенные голоса macOS, машину не покидает вообще
ничего.

Есть ещё два служебных запроса: раз в день приложение спрашивает у публичного API
GitHub, не вышел ли новый релиз (обычный HTTPS-запрос — никаких идентификаторов сверх
того, что несёт любой веб-запрос; отключается в «Настройки → О приложении»), и то же
самое делают кнопки скачивания на сайте.
Это вся сетевая история.

Демон `antiphond` слушает только `127.0.0.1` — из сети он недоступен. Реплики ваших
агентских сессий идут от плагина к демону и в приложение целиком по localhost.

## Что хранится на диске

Всё, что хранит Antiphon, лежит в `~/.antiphon`:

- `config.json` — настройки, включая API-ключи (права файла `0600`)
- `agents.json` — реестр агентов (id сессий, персоны, места)
- `tts-cache/` — синтезированный звук
- `antiphond.log` — диагностический лог демона, обнуляется при каждом запуске

Удалите каталог — и Antiphon вас забыл: см. [Удаление](./install.md#удаление).

## Этот сайт

Сайт статический (GitHub Pages). Кнопки скачивания запрашивают публичный API GitHub,
чтобы найти свежий релиз; веб-демо работает целиком в браузере и никуда ничего не
отправляет.
