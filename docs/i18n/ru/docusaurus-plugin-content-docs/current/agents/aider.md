# Aider

У Aider нет системы плагинов — только команда уведомления, которая срабатывает, без
полезной нагрузки, всякий раз, когда LLM закончил и ждёт ввода. Поэтому этот адаптер —
обёртка запуска с честной, но ограниченной полнотой: присутствие в комнате и мягкий толчок,
когда Aider вас ждёт.

## Установка

```bash
cp plugins/aider/antiphon-aider ~/bin/ && chmod +x ~/bin/antiphon-aider
antiphon-aider  # вместо `aider`, те же аргументы
```

## Что вы получаете

- **Привязка** при запуске обёртки — сессия занимает место.
- **«Ждёт вас»** — уведомление Aider отображается в цветение блокировки. Поскольку Aider
  не отличает *готово* от *ждёт подтверждения*, каждый конец хода подталкивает вас; это
  потолок API, а не баг.
- **Готово** при выходе обёртки (коды выхода сохраняются).

Ни тиков инструментов, ни нарратива от модели; обратный канал — только через инъекцию в
панель терминала. Если `aider` не обёрнут Antiphon или демон не запущен, обёртка делает
`exec` в aider напрямую — нулевая цена.
