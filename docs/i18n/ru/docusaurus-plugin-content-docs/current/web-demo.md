# Веб-демо

[antiphon.dev/demo.html](https://antiphon.dev/demo.html) — это полный движок: то же
DSP-ядро на Rust, скомпилированное в WebAssembly, рендерит ту же комнату, с трекингом
головы через камеру браузера (MediaPipe вместо нативного трекера). Это не видеоролик и не
приближение: нативная и wasm-сборки движка проверены на побайтовую идентичность в
паритетных тестах уровня CI.

Чем демо отличается от приложения:

- **Агенты играют по сценарию.** Четыре персоны проживают правдоподобную рабочую сессию —
  тики инструментов, бормотание, завершения — вместо ваших настоящих сессий.
- **Нет обратного канала.** Для ответов агентам нужен демон на вашей машине.
- **Калибровка упрощена**, хотя onboarding с «замрите на секунду» тот же самый.

Это честный способ оценить пространственный рендеринг, ничего не устанавливая: если голоса
в наушниках для вас не экстернализуются, подвигайте ползунок подстройки в onboarding — а
если не помогло и это, приложение звучать иначе не будет.

Chrome или Safari, наушники обязательны, камера — по желанию (от трекинга можно отказаться
и управлять мышью).
