# Antiphon 是什么？

Antiphon 是一台为编程智能体打造的空间音频监视器。你运行的每一个智能体会话，都会获得一个声音，以及在你周围虚拟房间里的一个位置。戴上耳机，你面对的不再是一整墙终端，而是一间可以*侧耳旁听*的工坊：左边有个智能体正在重写你的鉴权流程，低声絮语；身后另一个在起草文档，轻轻哼鸣；当某个智能体完成了工作，它会开口告诉你——声音正来自它干活的那个方位。

这个名字本身就是主旨：*ἀντίφωνον*——此起彼应的歌声，隔着空间彼此应答的声音。

## 工作原理

三个部分，全部在[同一个仓库](https://github.com/cfoust/antiphon)里，全部 MIT 许可：

- **Antiphon.app**——原生 macOS 应用。它以双耳（binaural）方式渲染整个房间（真实的 HRTF 卷积、房间声学、通过摄像头实现的头部追踪），以俯视雷达图显示你的智能体，并允许你通过键入回复它们——配合语音输入工具，也可以直接开口说。
- **antiphond**——一个小型本地守护进程，随应用打包。智能体插件连接到它；它为每个会话分配人格与房间中的座位，合成语音（ElevenLabs、OpenAI，或 macOS 内置语音），并把一切流式传给应用。它只监听 `127.0.0.1`。
- **插件**——为编程智能体准备的轻量适配器。目前支持 [Claude Code、Codex、OpenCode、Pi 和 Aider](./agents/index.md)，协议是开放的——写一个适配器只需几百行代码。

## 一切的核心

Antiphon 围绕一个交互而建：**闭上眼睛**。

睁眼时，房间不会打扰你——智能体只是安静的存在：工具运行时是一声轻柔的和弦，专注干活的那位则是一阵低沉的机器嗡鸣。当你闭上眼睛，摄像头察觉到了，房间便在你周围显现。把头转向某个声音，稍作停留，那个刚完成任务的智能体就会告诉你它做了什么。睁开眼，你回到桌前，回复框已经在等你。

没有提示音，没有闪烁。等待中的智能体只会在你的一侧耳畔缓缓积起一片和声，直到你有空理会它。

先在[网页演示](https://antiphon.dev/demo.html)里体验这种感觉，然后[安装应用](./install.md)。
