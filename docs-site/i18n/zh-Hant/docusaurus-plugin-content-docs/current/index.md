# Antiphon 是什麼？

Antiphon 是一款為程式代理（coding agent）打造的空間音訊監測器。你執行的每一個代理工作階段，都會獲得一個聲音，以及在你身邊虛擬房間裡的一個位置。戴上耳機，取代滿牆終端機的，是一間你可以*側耳旁聽*的工作坊：正在重整你驗證流程的代理在你左邊低語，另一個在你身後安靜地哼著寫文件，而當某個代理完成工作時，它會告訴你——聲音正來自它工作的所在。

名字本身就是重點：*ἀντίφωνον*（對唱）——聲音，在空間中彼此應答。

## 運作方式

三個部分，全部收在[同一個儲存庫](https://github.com/cfoust/antiphon)，全部採用 MIT 授權：

- **Antiphon.app** — 原生 macOS 應用程式。它以雙耳（binaural）方式渲染整個房間（真實的 HRTF 卷積、室內聲學、透過網路攝影機進行頭部追蹤），顯示一張代理的俯視雷達圖，並讓你以打字回覆代理——或者搭配語音輸入工具，用說的也行。
- **antiphond** — 一個小巧的本機常駐程式（daemon），隨附在應用程式裡。代理外掛連上它之後，它會為每個工作階段指派一個角色與房間裡的座位、合成語音（ElevenLabs、OpenAI，或內建的 macOS 語音），並把一切串流到應用程式。它永遠只監聽 `127.0.0.1`。
- **外掛** — 為各家程式代理準備的輕薄轉接器。目前支援 [Claude Code、Codex、OpenCode、Pi 與 Aider](./agents/index.md)，而且協定是開放的——寫一個轉接器只需要幾百行程式碼。

## 一切的核心

Antiphon 圍繞著一個互動而生：**「閉上眼睛」**。

睜著眼睛時，房間不會來打擾你——代理只是安靜的存在：工具執行時的一聲輕柔和弦，埋首工作者發出的低沉機械嗡鳴。當你閉上眼睛，網路攝影機會察覺，房間便在你四周浮現。把頭轉向某個聲音，停留片刻，那個完成工作的代理就會告訴你它做了什麼。睜開眼睛，你已回到書桌前，回覆框正等著你。

沒有任何提示音，沒有任何閃爍。等待中的代理只會在你的一側耳中，緩慢地醞釀出和聲的綻放，直到你有空理它。

先到[網頁展示](https://antiphon.dev/demo.html)感受一下，然後[安裝應用程式](./install.md)。
