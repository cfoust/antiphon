# 快速上手

你已经[安装了应用](./install.md)并接入了一个智能体。下面是 Antiphon 在日常使用中的样子。

## 房间

主窗口是一幅俯视雷达图：你是中心的瞳孔，智能体是围坐在你周围的圆点。拖动一个智能体即可移动它——它的声音随之而动，并且你会从新位置听到一声轻柔的声呐脉冲。座位在重启后依然保留。

右侧的列表显示每个会话：状态、最后一条消息，以及等待指示。悬停某一行，就会在房间里聚焦那个智能体。给某个智能体按下小睡（月亮图标），它会退出这个世界——但仍持续接收更新，唤醒时从上次的位置重新入座。

角落里的眼睛按钮是总开关。闭眼状态：摄像头释放、输出静默、窗口变暗。睁眼状态：观察并聆听。

## 你会听到什么

每个智能体都有一套音乐身份——以它的提示音为根音构成的和弦：

- **工具滴答。**每次工具调用都会拨响该智能体下行和弦的下一个音。不用看屏幕，你就知道谁在忙。
- **工作嗡鸣。**深入长任务的智能体会沉入其音域底部——一阵低沉的三音机器嗡鸣。
- **等待之花。**完成了工作、想找你的智能体，会在你的一侧耳畔缓缓积起一片小七和弦的和声——以分钟计，而非毫秒。它从不催促；它只是慢慢积累。
- **语音。**智能体完成某件事时会开口说出来——一两句话，用它自己的声音，从它的座位上传来。

## 闭上眼睛

摄像头追踪你的头部姿态和眼睛状态（全部在本机完成——任何数据都不会离开你的电脑）。

睁眼时，语音被暂时按下，你只会听到环境声层。**闭上眼睛，房间便显现出来**：短暂的淡入之后，每个智能体都出现在其圆点所在的准确位置。把头转向其中一个，稳住片刻——一声轻微的顶峰音标记锁定，如果它有消息，就会送上它的总结。再转向下一个声音；如此往复。这是用耳朵完成的分诊，几秒钟即可。

睁开眼，世界退去。如果有智能体说了话，回复框已在等待：键入（或口述）后按回车，你的话就会注入那个智能体的会话。

## 设置

齿轮按钮在窗口内打开设置：

- **适配**——一个频率缩放滑块，用于让 HRTF 匹配你的耳朵；调整时会有一段循环语音从正前方播放。只需设置一次，调到那个声音听起来最*在你脑袋之外*的位置。
- **追踪**——摄像头选择与重新校准。
- **语音**——TTS 提供商、API 密钥，以及逐一开关的语音。
- **语言**——English、Русский、简体中文、繁體中文。
