# 网页演示

[antiphon.dev/demo.html](https://antiphon.dev/demo.html) 是完整的引擎——同一个 Rust DSP 核心编译成 WebAssembly，渲染同一个房间，并通过浏览器摄像头进行头部追踪（用 MediaPipe 替代原生追踪器）。它不是视频，也不是近似品；引擎的原生构建与 wasm 构建在 CI 级别的一致性测试中被验证为逐字节一致。

与应用的区别：

- **智能体是脚本化的。**四个角色演绎一段逼真的工作会话——工具滴答、低语、完成播报——而不是你真实的会话。
- **没有对话回传。**回复流程需要在你的机器上运行守护进程。
- **校准做了简化**，不过"保持静止"的引导流程是一样的。

在安装任何东西之前，这是评估空间渲染最诚实的方式：如果戴耳机时声音在你听来无法外化，请在引导流程中调整适配滑块——如果调整后仍然不行，应用也不会有什么不同。

需要 Chrome 或 Safari，必须戴耳机，摄像头可选（你可以拒绝追踪，改用鼠标来操控视角）。
